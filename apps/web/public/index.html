A simple registry for Vite React components

Usage

# Install CLI
bun install -g buildy

# Initialize in your Vite project
buildy init

# Add components
buildy add button card

# Add from external URL
buildy add "https://buildy.tw/r/ui/button.json"

Available Components

  • Button - Customizable button component

Registry API

GET /r/index.json - List all components
GET /r/ui/{component}.json - Get specific component
